# @onersoft Impeccable Design Brief

## Design Context

### Users
일반 소비자 웹 서비스 사용자. @onersoft 프로덕트의 랜딩, 콘텐츠 소비, 가벼운 트랜잭션 등 보편적인 웹 인터랙션을 수행.

### Brand Personality
**Refined · Calm · Precise.** 장식적이지 않고, 조용하지만 의도적. 모든 픽셀이 이유를 가진다.

### Aesthetic Direction
- 라이트/다크 모두 1급 시민. 기본 테마는 `system` 선호.
- 브랜드 hue는 청회색 계열(`oklch(0.69 0.05 230)` 기반). 채도 상승 금지 — calm 톤 유지.
- 타이포그래피는 system-ui stack + 한국어 Pretendard fallback. 호스트 앱이 `--ds-font-sans`를 override해 브랜드 서체를 입힐 수 있도록 설계.
- Accent는 60-30-10 규칙의 10 영역에만. 자주 쓰지 않는다.
- Neutral은 브랜드 hue 쪽으로 미세 tint되어 잠재적 응집감 확보.

### Anti-references
- 그라디언트 텍스트, 그라디언트 배경 장식
- border-left/right >= 2px 의 "accent stripe"
- 글래스모피즘(backdrop-filter blur) 장식
- 네온 글로우 / 채도 높은 shadow
- Pure `#000` / `#fff` 전경·배경
- Bounce/elastic easing
- 큰 rounded icon + heading 템플릿

### Design Principles
1. 장식보다 정밀도. 표현은 여백·타이포 계층·모션 타이밍으로.
2. Neutral은 모두 브랜드 hue 쪽으로 미세 tint되어 잠재적 응집감을 만든다.
3. Accent는 드물게. 자주 나오면 힘을 잃는다.
4. 상호작용은 즉각적으로, 감정적 과장 없이. Easing은 단조 감속(ease-out 계열)만.
5. 라이트/다크에서 동일한 감각을 주되 동일한 색을 쓰지 않는다.
