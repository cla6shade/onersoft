## Usage

```tsx
import { Toggle } from '@onersoft/design-system'

<Toggle pressed={bold} onPressedChange={setBold} aria-label="Toggle bold">B</Toggle>
```

## Props

| Prop | Type | Default |
| ---- | ---- | ------- |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |
| `pressed` / `defaultPressed` | `boolean` | — |
| `onPressedChange` | `(pressed) => void` | — |
| `disabled` | `boolean` | `false` |
