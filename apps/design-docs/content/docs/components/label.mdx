## Usage

```tsx
import { Label, Input } from '@onersoft/design-system'

<Label htmlFor="email">Email</Label>
<Input id="email" type="email" />
```

## Notes

`Label` wraps Radix [`Label.Root`](https://www.radix-ui.com/primitives/docs/components/label) and inherits its props. It connects to controls via `htmlFor` (matching the control's `id`) or implicit nesting.
