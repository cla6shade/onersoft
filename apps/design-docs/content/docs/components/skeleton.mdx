## Usage

```tsx
import { Skeleton } from '@onersoft/design-system'

<Skeleton height={120} />
<Skeleton variant="text" width="60%" />
<Skeleton variant="circle" width={40} />
```

## Props

| Prop | Type | Default |
| ---- | ---- | ------- |
| `variant` | `'block' \| 'text' \| 'circle'` | `'block'` |
| `width` | `number \| string` | — |
| `height` | `number \| string` | — |
