## Usage

```tsx
import { Accordion } from '@onersoft/design-system'

<Accordion.Root type="single" collapsible>
  <Accordion.Item value="item-1">
    <Accordion.Header>
      <Accordion.Trigger>Heading</Accordion.Trigger>
    </Accordion.Header>
    <Accordion.Content>Body</Accordion.Content>
  </Accordion.Item>
</Accordion.Root>
```

## Props

`Accordion.Root` mirrors Radix [`Accordion.Root`](https://www.radix-ui.com/primitives/docs/components/accordion#root). Most-used:

| Prop | Type | Default |
| ---- | ---- | ------- |
| `type` | `'single' \| 'multiple'` | — |
| `collapsible` | `boolean` | `false` |
| `defaultValue` | `string \| string[]` | — |
| `value` | `string \| string[]` | — |
| `onValueChange` | `(value) => void` | — |
