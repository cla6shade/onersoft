## Props

| Prop | Type | Default |
| ---- | ---- | ------- |
| `variant` | `'neutral' \| 'accent' \| 'success' \| 'warning' \| 'danger'` | `'neutral'` |
| `size` | `'sm' \| 'md'` | `'md'` |
| `asChild` | `boolean` | `false` |
